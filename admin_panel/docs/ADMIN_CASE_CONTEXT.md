# Admin Panel Context

This panel is a supervisor/admin interface for monitoring the results of a mobile industrial inspection app used by field workers.

## Product Goal

Provide a clean Stage 1 web dashboard where supervisors can:

- view inspection tasks
- monitor completion status
- review equipment inspection results
- review defect reports
- view photo attachments
- see recent activity
- get a quick dashboard summary

## Scope

This is only a Stage 1 supervisor panel prototype.  
It is not a full enterprise ERP, SCADA, or IoT platform.

## Core Modules

- Dashboard
- Tasks
- Task Details
- Equipment
- Defects
- Reports
- Settings placeholder

## Language Rule

**All visible product UI must be in Russian.**  
Code structure (identifiers, filenames, internal variable names) may remain in English.

## Technical Notes

- use mock data only
- keep the structure ready for future API integration
- do not overengineer
- keep it demo-ready and presentable
