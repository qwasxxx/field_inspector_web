# field_inspector_web

В репозитории фронтенд живёт в **`frontend/`** (Vite + React + TypeScript).

```bash
cd frontend
npm install
npm run dev
```

Подробности, Supabase, маршруты и архитектура: [`frontend/README.md`](frontend/README.md).

Стандарты кода фронтенда: [`frontend/docs/CODING_STANDAERTS.md`](frontend/docs/CODING_STANDAERTS.md).

Переменные окружения (например `VITE_SUPABASE_*`) задаются **только** в `frontend/.env.local` (см. `frontend/.env.example`).
